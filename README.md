# CRM App - Cliente y Notas

Una aplicación CRM simple y eficiente desarrollada con HTML y JavaScript vanilla para gestionar clientes y sus notas asociadas.

## 📋 Descripción

Esta aplicación permite administrar una base de datos de clientes con la capacidad de agregar, editar y eliminar información de contacto, así como mantener un registro de notas por cada cliente.

## ✨ Características

- **Gestión de Clientes**: Crear, editar y eliminar perfiles de clientes
- **Sistema de Notas**: Agregar notas específicas para cada cliente
- **Interfaz Intuitiva**: Diseño limpio y fácil de usar
- **Almacenamiento Local**: Los datos se guardan en el navegador
- **Responsivo**: Compatible con dispositivos móviles y escritorio
