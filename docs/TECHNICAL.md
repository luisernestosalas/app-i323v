## Funcionalidades Principales

### Gestión de Clientes
- **Crear cliente:** Registro de nuevos clientes con información básica
- **Editar cliente:** Modificación de datos existentes
- **Eliminar cliente:** Borrado de registros de clientes
- **Listar clientes:** Visualización de todos los clientes registrados

### Gestión de Notas
- **Crear nota:** Añadir notas asociadas a clientes
- **Editar nota:** Modificar contenido de notas existentes
