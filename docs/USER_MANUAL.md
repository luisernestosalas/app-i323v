# Manual de Usuario - CRM con Clientes y Notas

## ¿Qué es esta aplicación?

Es un sistema simple para gestionar clientes y sus notas. Puedes agregar información de clientes y escribir notas sobre cada uno.

## Cómo acceder

Abre tu navegador web y ve a:

https://app-i323v-h3jkgv18t-luisernestosalas-2775s-projects.vercel.app


## Funciones principales

### 1. Agregar un cliente nuevo
- Busca el botón "Agregar Cliente" o "+"
- Completa los campos:
  - Nombre
  - Teléfono
  - Email
  - Empresa (opcional)
- Haz clic en "Guardar"

### 2. Ver la lista de clientes
- En la pantalla principal verás todos tus clientes
- Cada cliente muestra su información básica

### 3. Buscar clientes
- Usa la barra de búsqueda en la parte superior
- Escribe el nombre, empresa o email
- Los resultados aparecen automáticamente

### 4. Ver detalles de un cliente
- Haz clic sobre cualquier cliente de la lista
- Verás toda su información y notas

### 5. Agregar notas
- Dentro de un cliente, busca "Agregar Nota"
